## Herramientas

Se debe contar con las siguientes herramientas:

- GIT
- Node
- Editor de código (Recomendado Visual Studio Code)

---

### Instalación

- Clonar este repositorio con el comando: `git clone https://github.com/MetalDummy/MLC_Back`.
- Instalar las librerías del proyecto con el comando:
```shell
$ npm install
```
- Iniciar el BackEnd con el comando:
```shell
$ node app.js
```
